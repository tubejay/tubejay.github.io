## Method

- step
  - corner
  - edge
    - bottom layer
      - move front right edge to front bottom edge : ( R' L ) ( R L' )
      - move front left edge to front bottom edge : ( L R' ) ( L' R )
    - middle layer
      - permute 3 edges
        - clockwise : ( R' U' ) ( R U' ) ( R' U' ) ( R )
        - counterclockwise : ( L U ) ( L' U ) ( L U ) ( L' )
      - flip 2 front edges : ( R' L ) ( R L' ) ( U L' ) ( U' L )
