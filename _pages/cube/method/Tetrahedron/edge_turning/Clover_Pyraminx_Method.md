## Shape

### Edge

### Petal

- Flip Two Petals : on F Face & R Edge
  - ( R' D ) ( R D' )

### Hidden Piece

- Move 1 Hidden Piece : on F Face to R Corner
  - ( R' D R D' ) x 3



## Color

### Petal

- Permute 3 Petals : on U Corner
  - Rotate B Edge
  - Permute 3 Petals
    - Clockwise : ( R' L ) ( R L' )
    - Counterclockwise : ( L R' ) ( L' R )
  - Rotate B Edge

### Center

- Swap 2 Centers : on F Face & R Face
  - Rotate Edge : on L Face & D Face
  - ( L' B L B' ) x 3
  - Rotate Edge : on L Face & D Face
