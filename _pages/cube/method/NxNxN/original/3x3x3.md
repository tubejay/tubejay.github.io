## CFEC

- step
  - Cross
  - F2L : First Two Layers
  - ELL : Edges of the Last Layer
    - EOLL : Edge Orientation of the Last Layer
      - [3x3x3 / CFOP / EOLL](#cfop)
    - EPLL : Edge Permutation of the Last Layer
      - permute 3 edges
        - clockwise except back edge : ( R' U' ) ( R U' ) ( R' U2 ) ( R )
