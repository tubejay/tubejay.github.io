## Method

- [Reduction](/cube/method/NxNxN/original/5x5x5/reduction)
  - Center
  - Edge
    - 5x5x5 / Edge / Parity
      - 1-1-1 Parity
  - 3x3x3 / Method
- [Yau](/cube/method/NxNxN/original/5x5x5/yau)
  - 2 Centers
  - 3 Cross Edges
  - 4 Centers
  - 1 Cross Edge
  - 8 Edges
    - 5x5x5 / Edge / Parity
      - 1-1-1 Parity
  - 3x3x3 / Method
    - Skip : Cross
- [Hoya](/cube/method/NxNxN/original/5x5x5/hoya)
  - 4 Centers
  - 4 Cross Edges
  - 2 Centers
  - 4x4x4 / Yau
    - Start : 8 Edges
- [LBL](/cube/method/NxNxN/original/5x5x5/lbl)
  - FL
    - Center
    - Edge
    - Cross
    - Corner
  - ML
    - Center
    - Edge
  - LL
    - Edge
      - 5x5x5 / Edge / Parity
        - 1-1-1 Parity
    - 3x3x3 / CFOP
      - Start : OLL
