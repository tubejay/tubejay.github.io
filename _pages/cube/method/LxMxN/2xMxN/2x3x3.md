## Method

### Corner

- Bottom Layer
  - Move Front Right Corner
    - from Top to Bottom
    - ( R2 U ) ( R2 U' ) ( R2 )
- Top Layer
  - Swap Corners
    - Front Right & Back Right
    - ( R2 U R2 U' R2 ) ( F2 U' F2 ) ( D R2 D')

### Edge

- Bottom Layer
  - Move Right Edge
    - from Top to Bottom
    - ( R2 U' ) x 6
- Top Layer
  - Swap Edges
    - Front Edge & Back Edge
    - ( R2 U2 ) x 3
  - Swap Edges
    - Front Edge & Right Edge
    - ( R2 U R2 U ) ( R2 U2 R2 U2 ) ( R2 U R2 U' ) ( R )
